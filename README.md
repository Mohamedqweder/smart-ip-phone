# Smart IP Phone

Schaltplan und PCB-Design für ein WLAN-basiertes IP-Telefon.
Entwickelt mit KiCad auf Basis des ESP32-WROOM-32.

## Dateien
- `Arbeit.kicad_sch` → Schaltplan
- `arbeit fertig.kicad_pcb` → PCB Layout
- `README_SmartIPPhone.pdf` → Vollständige Dokumentation

## Design
- Mikrocontroller: ESP32-WROOM-32
- Konnektivität: WLAN 802.11 b/g/n (VoIP/SIP)
- Tool: KiCad
